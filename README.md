# Car Object Detection using ResNet50 (CNN)

A deep learning project that detects cars in images using a Convolutional Neural Network (CNN) with transfer learning based on ResNet50.
This project also compares different optimization algorithms to evaluate performance.

##  Features
- Transfer Learning (ResNet50)
- Data Augmentation
- Optimizer Comparison (Adam, SGD, RMSprop)
- Model Checkpointing
- Accuracy Visualization
- Results Logging
